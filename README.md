# invoice-generator
---
An Invoice generating application

**Tech :** 
1. Library : React Js
2. CSS Framework : TailwindCSS

### Installation
---

* Clone this repository
* `npm install -D parcel`
* `npm install react`
* `npm install react-dom`
* Install & setup Tailwindcss with parcel (as per docs)
* Finally, Run `npm start`
